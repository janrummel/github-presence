# Lernreise / Kurs

## README

### Bronze
- problem-statement: Was lernst Du? (Lernziele)
- audience: Fuer wen? (Zielgruppe + Voraussetzungen)

### Silber
- curriculum: Curriculum / Inhaltsverzeichnis
- entry-point: Wie starten? (Einstiegspunkt)
- license: Lizenz + Beitragen

### Gold
- learning-outcomes: Konkrete Lernziele pro Kapitel/Level
- prerequisites-detail: Voraussetzungen detailliert (Tools, Vorkenntnisse)
- faq: FAQ in README

### Platin
- cross-references: Cross-References zu anderen Lernressourcen
- community: Community/Diskussions-Link
