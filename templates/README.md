# README Templates

One template per project type. Each follows the Gold-tier checklist and uses `{{PLACEHOLDER}}` syntax for variable content.

| Template | Project Type | Based On |
|----------|-------------|----------|
| [cli-tool-readme.md](cli-tool-readme.md) | CLI Tool / Dev Tool | CloakCode (Gold reference) |
| [lernreise-readme.md](lernreise-readme.md) | Learning Path / Course | Level Up AI + AI Product Thinking (Gold references) |
| [framework-readme.md](framework-readme.md) | Framework / Starter Kit | Project type definitions |
| [checklist-readme.md](checklist-readme.md) | Checklist / Content Project | Project type definitions |
| [mini-tool-readme.md](mini-tool-readme.md) | Mini Tool / Experiment | Project type definitions |

## Usage

1. Pick the template matching your project type
2. Copy it to your repo as `README.md`
3. Replace all `{{PLACEHOLDER}}` values
4. Remove sections marked as optional that don't apply
5. Run `/website-review` to verify your score
